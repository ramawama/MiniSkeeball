# Skeeball

Skeeball game implementation for the micro:bit v2.

# Description
Building a mini skeeball game on micro:bit v2 (nRF52833) with 12mm steel ball bearings.
IR break beams detect scoring per ring, APDS9960 handles gesture-based menu navigation,
and a 0.96" 16-bit color OLED (SPI) displays scores and menus.

Uses a 3D printed skeeball machine
